## Project Description

This project is a **Telegram chatbot** created to help Kyrgyz-speaking users with:

- Study counseling for universities in Germany  
- Learning the German language with explanations in Kyrgyz  

The chatbot aims to make information about **studying and living in Germany**
more accessible and easier to understand.

## Target Audience

- Kyrgyz-speaking students who want to study in Germany  
- People looking for guidance on:
  - Universities and higher education institutions  
  - Application processes and requirements  

## Features

- Telegram-based chatbot  
- Study counseling:
  - General information about studying in Germany  
  - Universities and higher education institutions  
  - Application process and requirements  
- German language learning:
  - Vocabulary explanations  
  - Grammar explanations in Kyrgyz  
  - Simple examples and dialogues  
- Multilingual interaction (Kyrgyz / German)

- Python  
- Telegram Bot API
